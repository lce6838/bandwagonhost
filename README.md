## 搬瓦工的核心优势是什么：CN2 GIA 线路

很多人买 VPS 不在乎线路，直到晚高峰的时候才发现打开一个网页要等好几秒，或者 SSH 连接卡到想摔键盘。

搬瓦工在中国用户里口碑这么好，核心就是 **CN2 GIA 线路**。

稍微解释一下：CN2 是中国电信旗下的优质国际专线，分两档——CN2 GT（半程优化）和 CN2 GIA（全程优化）。GIA 是 Global Internet Access 的缩写，全程走 AS4809，不经过拥堵的 163 骨干网，是电信等级最高的国际线路，价格也最贵。

实测下来，搬瓦工 CN2 GIA-E 机房在晚高峰期间延迟稳定在 158ms 左右，丢包率接近 0，而同价位的普通线路 VPS 有时候丢包能到 30%+。

所以不是说搬瓦工配置多强，而是在这个价位段，**能稳定提供 CN2 GIA 线路的选择真的不多**。
